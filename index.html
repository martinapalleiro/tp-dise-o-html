La base de HTML

La base de html, may 12, 2024

HTML es un lenguaje utilizado para crear páginas web. Se compone de etiquetas que definen la estructura y contenido de una página. Las etiquetas se utilizan para organizar el contenido, agregar imágenes, enlaces y otros elementos multimedia, así como para dar formato al texto y establecer estilos visuales. Las páginas HTML se pueden ver en un navegador web una vez creadas.

Estructura basica

Estructura basica,may 12, 2024

La estructura básica de un documento HTML consta de las siguientes partes:

    < !DOCTYPE html >: Declaración del tipo de documento, que indica al navegador que se trata de un documento HTML5. < html >: Elemento raíz que engloba todo el contenido HTML de la página. < head >: Contiene metadatos sobre el documento, como el título de la página, enlaces a hojas de estilo (CSS), y otros elementos que no se muestran directamente en la página. < title >: Define el título de la página que se muestra en la barra de título del navegador. < body >: Contiene todo el contenido visible de la página, como texto, imágenes, enlaces, etc
